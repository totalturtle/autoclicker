# 안드로이드 오토클리커 앱 기능 초안

## 개요
Android Accessibility Service를 활용한 자동 터치/클릭 앱

---

## 핵심 기능

### 1. 클릭 포인트 설정
- 좌표 수동 입력 (X, Y)
- **터치로 좌표 직접 선택** ✅ 구현됨
  - 포인트 추가 다이얼로그에서 "터치로 시작 좌표 선택" 버튼 클릭
  - 반투명 전체화면 오버레이(CoordPickerService) 표시
  - 원하는 위치 터치 → 앱 복귀 + 다이얼로그 자동 재오픈 후 X/Y 자동 입력
  - 스와이프 끝 좌표 / 트리거 확인 좌표도 동일 방식으로 터치 선택 가능
- 다중 포인트 등록 (최대 50개)
- 포인트별 이름/라벨 지정

### 2. 클릭 시퀀스 (순서 제어)
- 등록된 포인트를 순서대로 반복 클릭
- 포인트 순서 드래그앤드롭으로 변경 ✅ 구현됨
- 반복 횟수 설정 (1회 ~ 무한) ✅ 구현됨
- 전체 시퀀스 반복

### 3. 타이밍 제어
- 각 클릭 간 딜레이 설정 (ms 단위) ✅ 구현됨
- 포인트별 개별 딜레이 설정 ✅ 구현됨
- 포인트별 랜덤 오차 설정 (탐지 방지용) ✅ 구현됨
  - 각 포인트 딜레이 기준 ±N ms 범위 내 랜덤 (0이면 비활성)
  - 예: 딜레이 1000ms, 오차 100ms → 900~1100ms 랜덤
  - 포인트별 설정값 우선, 미설정 시 전역 딜레이 사용
- 클릭 지속 시간(Long Press) 설정 ✅ 구현됨

### 4. 제스처 지원
- 단순 탭 (Tap) ✅ 구현됨
- 롱 프레스 (Long Press) ✅ 구현됨
- 스와이프 (Swipe) — 시작점 → 끝점 ✅ 구현됨
- 드래그 (Drag) — 미구현
- 핀치 줌 인/아웃 — 미구현

### 5. 시작/정지 제어
- 플로팅 버튼으로 언제든 시작/정지 ✅ 구현됨
- 물리 버튼(볼륨 다운 키)으로 시작/정지 토글 ✅ 구현됨
- 알림바 Quick Tile로 제어 ✅ 구현됨 (타일 시각 상태 반영 포함)
- 횟수 조건 충족 시 자동 정지 ✅ 구현됨

### 6. 프로필 관리 ✅ 구현됨
- 다중 named 프로필 저장/불러오기 (SharedPreferences)
- 프로필 이름 지정
- 프로필 내보내기/가져오기 (JSON, SAF 파일 피커)
- 앱별 프로필 자동 연동 (특정 앱 포그라운드 전환 시 자동 로드)

### 7. 조건부 실행 (스마트 트리거) ✅ 부분 구현
- 특정 색상 감지 시 클릭 실행 ✅ 구현됨 (API 30+ / Android 11 이상)
  - 조건 불충족 시: 스킵 or 재시도(N회) 선택 가능
  - 허용 오차(tolerance) 설정 가능
- 특정 이미지/아이콘 인식 후 클릭 (OpenCV) — 미구현
- 텍스트 인식(OCR) 후 조건 분기 (ML Kit) — 미구현

### 8. 화면 오버레이 UI ✅ 구현됨
- **플로팅 컨트롤 패널** (항상 화면 위에 표시, 권한 확보 시 앱 시작과 함께 자동 실행)
  - `· · ·` 핸들로 패널 드래그 이동
  - ⚙ 설정 버튼 → 메인 앱 열기
  - **+ 버튼** → 화면에 드래그 가능한 포인트 마커 생성 (번호 순)
  - **- 버튼** → 마지막 포인트 마커 삭제
  - ▶ / ■ 시작 / 정지
  - 클릭 횟수 실시간 표시
- **드래그 가능한 포인트 마커** ✅ 구현됨
  - + 누를 때마다 화면에 번호가 표시된 마커 생성
  - 마커를 드래그해 원하는 위치로 이동 → 좌표 자동 저장
  - 마커 탭 → 세부 설정 팝업 (PointSettingsActivity)
    - 제스처 변경 (탭/롱프레스/스와이프)
    - 딜레이, 랜덤 오차(±ms), 레이블, 트리거 설정
    - 개별 포인트 삭제
  - 👁 눈 아이콘으로 마커 표시/숨김 토글 (OverlayService 내장)
  - 자동 클릭 실행 중에는 마커 자동 숨김, 정지 후 복원
- 자동 클릭 실행 중에도 패널 유지

## 보조 기능

### 9. 통계 및 로그 ✅ 구현됨
- 세션별 총 클릭 횟수 / 스킵 횟수 / 경과 시간 기록
- 인메모리 로그 (타임스탬프 + 이벤트 메시지)
- 로그 파일 내보내기 (SAF .txt 저장)

### 10. 설정 ✅ 구현됨
- 다크/라이트/시스템 기본 테마 선택
- 앱 시작 시 저장된 테마 즉시 적용 (Application 클래스)
- Accessibility Service 권한 안내 및 바로가기
- 앱 버전 정보 표시
- 언어: 한국어/영어 (OS 언어 자동 감지)

### 11. 첫 실행 가이드 ✅ 구현됨 (UX 개선)
- 최초 실행 시 권한 설정 단계별 안내 팝업
- 접근성 권한 버튼 클릭 시 상세 안내 다이얼로그
  ("설치된 앱 → AutoClicker 자동 클릭 → 스위치 ON")
- 오버레이 권한 버튼 클릭 시 상세 안내 다이얼로그

## 권한 요구사항
| 권한 | 용도 |
|------|------|
| `BIND_ACCESSIBILITY_SERVICE` | 자동 터치 실행 핵심 권한 |
| `SYSTEM_ALERT_WINDOW` | 플로팅 오버레이 표시 |
| `FOREGROUND_SERVICE` | 백그라운드 실행 유지 |
| `FOREGROUND_SERVICE_SPECIAL_USE` | 포그라운드 서비스 타입 |
| `RECEIVE_BOOT_COMPLETED` | 부팅 후 자동 시작 (미구현) |

## 기술 스택
- **언어**: Kotlin
- **최소 SDK**: API 26 (Android 8.0)
- **타겟 SDK**: API 36
- **UI**: XML + ViewBinding
- **서비스**: AccessibilityService, ForegroundService
- **색상 감지**: AccessibilityService.takeScreenshot() (API 30+)
- **이미지 인식**: 미구현 (OpenCV 검토)
- **OCR**: 미구현 (ML Kit 검토)
- **저장**: SharedPreferences + JSON 파일 (SAF)
- **테마**: AppCompatDelegate DayNight

## 화면 구성

```
[메인 화면]
 ├─ 권한 상태 카드 (접근성 / 오버레이)
 ├─ 빠른 시작/정지 카드 (볼륨키 토글 스위치, QS Tile 안내)
 ├─ 클릭 포인트 카드
 │   ├─ 포인트 목록 (RecyclerView, 드래그 순서변경)
 │   ├─ 포인트 추가 버튼
 │   ├─ 눈 아이콘 (마커 표시/숨김 토글)
 │   ├─ 공통 딜레이 입력
 │   └─ 반복 횟수 입력
 ├─ 실행 상태 카드 (상태 텍스트, 클릭 횟수)
 ├─ 유틸리티 버튼 (프로필 관리 | 통계/로그 | 설정)
 └─ 시작 / 정지 버튼

[포인트 추가 다이얼로그]
 ├─ 제스처 선택 (탭 / 롱프레스 / 스와이프)
 ├─ 좌표 / 끝좌표 입력
 ├─ 지속시간 / 딜레이(ms) / 랜덤 오차(±ms) 입력
 └─ 색상 트리거 설정 (선택)
     ├─ 확인 좌표 X, Y
     ├─ 목표 색상 (#RRGGBB)
     ├─ 허용 오차
     └─ 불일치 시 동작 (스킵 / 재시도)

[플로팅 오버레이]
 ├─ ▶ 시작 / ■ 정지 버튼 (드래그 이동 가능)
 └─ 현재 클릭 횟수 텍스트

[프로필 관리]
 ├─ 현재 설정 저장 버튼
 ├─ 파일에서 가져오기 버튼
 └─ 프로필 목록 (불러오기 / 내보내기 / 앱연동 / 삭제)

[통계 및 로그]
 ├─ 세션 시작 시각 / 경과 시간
 ├─ 총 클릭 수 / 스킵 수
 ├─ 로그 내보내기 버튼
 └─ 타임스탬프 로그 (monospace)

[설정]
 ├─ 테마 선택 (시스템 / 라이트 / 다크)
 ├─ 접근성 서비스 바로가기
 └─ 앱 버전 정보
```

| Phase 4 | 색상 트리거 (API 30+), 세션 통계 및 로그 | ✅ 완료 |
| Phase 5 | 랜덤 딜레이, 다크/라이트 테마, 영어 지원, 릴리스 빌드 설정 | ✅ 완료 |
| 버그수정 | QS Tile 시각 상태, OverlayService JSON 동기화, 중복 STOP 제거 | ✅ 완료 |
| UX개선 | 첫 실행 가이드, 권한 버튼 단계별 안내 다이얼로그 | ✅ 완료 |
| UX개선 2 | 터치로 좌표 선택 (CoordPickerService 오버레이, 다이얼로그 상태 복원) | ✅ 완료 |

| 좌표 시각화 | OverlayService 내장 마커 (번호 표시), CoordOverlayService 제거 (중복 마커 버그 해결) | ✅ 완료 |

| 동작 개선 | dispatchGesture 콜백 기반 suspend 처리 (제스처 완료 후 딜레이 적용), 실행 중 마커 자동 숨김 | ✅ 완료 |
| 랜덤 오차 재설계 | 전역 랜덤 최소/최대 제거 → 포인트별 ±오차(ms) 방식으로 변경, 포인트 설정값 우선 적용 | ✅ 완료 |
| 버그수정 2 | 재생 시 첫 번째 포인트 스킵 버그 수정 (코루틴 시작 전 300ms 초기 딜레이 추가) | ✅ 완료 |

## 알려진 제한사항
- 색상 감지는 Android 11(API 30) 이상에서만 동작
- 드래그, 핀치 줌 제스처 미구현
- 이미지/텍스트 인식 트리거 미구현
- Play 스토어 미등록 상태 (사이드로드 설치 시 Play Protect 경고 발생)
